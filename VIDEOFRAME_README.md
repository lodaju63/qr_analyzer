# 비디오 프레임 추출 도구 (videoframe.py)

## 📋 개요

영상 파일에서 원하는 프레임을 추출하고 저장할 수 있는 간단한 Streamlit 도구입니다.

## 🚀 실행 방법

```bash
streamlit run videoframe.py
```

## ✨ 주요 기능

### 1. 프레임 번호로 추출
- 특정 프레임 번호를 지정하여 추출
- 슬라이더로 프레임 번호 선택
- 해당 프레임의 시간(초)도 함께 표시

### 2. 시간(초)으로 추출
- 재생 시간(초)을 기준으로 프레임 추출
- 슬라이더로 시간 선택
- 자동으로 프레임 번호 계산

### 3. 여러 프레임 일괄 추출
- 시작 프레임부터 끝 프레임까지 일정 간격으로 추출
- 최대 추출 개수 제한 설정 가능
- ZIP 파일로 일괄 다운로드

### 4. 다운로드 옵션
- **단일 프레임**: 개별 이미지 파일로 다운로드
- **여러 프레임**: ZIP 파일로 일괄 다운로드

## 📦 지원 형식

- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)
- FLV (.flv)
- WMV (.wmv)

## 🎯 사용 예시

### 예시 1: 특정 시간의 프레임 추출
```
1. 비디오 파일 업로드
2. "시간(초)으로 추출" 선택
3. 슬라이더로 원하는 시간 선택 (예: 30.5초)
4. "프레임 추출" 버튼 클릭
5. 추출된 프레임 확인 및 다운로드
```

### 예시 2: 여러 프레임 일괄 추출
```
1. 비디오 파일 업로드
2. "여러 프레임 일괄 추출" 선택
3. 시작 프레임: 0
4. 끝 프레임: 1000
5. 간격: 30 (30프레임마다 추출)
6. 최대 추출 개수: 10
7. "여러 프레임 추출" 버튼 클릭
8. ZIP 파일로 다운로드
```

## 💡 활용 예시

### QR 코드 분석
- 비디오에서 QR 코드가 나타나는 특정 프레임 추출
- 프레임별로 QR 코드 상태 분석

### 이미지 데이터셋 생성
- 여러 비디오에서 일정 간격으로 프레임 추출
- 학습 데이터셋 생성

### 비디오 분석
- 특정 이벤트가 발생하는 프레임 추출
- 타임라인 기반 분석

## 📝 주의사항

1. **대용량 비디오**: 매우 긴 비디오의 경우 일괄 추출 시 시간이 걸릴 수 있습니다.
2. **메모리**: 많은 프레임을 동시에 추출하면 메모리 사용량이 증가합니다.
3. **파일 형식**: 일부 특수 코덱은 지원하지 않을 수 있습니다.

## 🎨 UI 구조

- **사이드바**: 비디오 업로드 및 정보 표시
- **메인 영역**: 프레임 추출 옵션 및 결과 표시
- **다운로드**: 추출된 프레임을 이미지 또는 ZIP으로 다운로드

## 📊 비디오 정보 표시

업로드된 비디오의 다음 정보를 표시합니다:
- 파일명
- 해상도 (가로 x 세로)
- FPS (초당 프레임 수)
- 총 프레임 수
- 재생 시간 (초)
